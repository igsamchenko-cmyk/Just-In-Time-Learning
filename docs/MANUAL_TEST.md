# Ручна перевірка MVP

1. Запусти сервер:

```powershell
uvicorn app.main:app --reload
```

2. Відкрий:

```text
http://127.0.0.1:8000
```

3. Натисни `Демо-вхід`.

4. Створи курс із запитом:

```text
Хочу навчитися за 2 дні налаштовувати деплой Python-скрипта на VPS через GitHub Actions.
```

5. Переконайся, що створилися картки модулів.

6. Відкрий перший модуль.

7. Введи коротку неправильну відповідь:

```text
Буду вчити тему.
```

Очікування: м'який фідбек і можливість спробувати ще.

8. Введи ще одну неповну відповідь.

Очікування: з'являється підказка.

9. Введи відповідь за структурою:

```text
Я спочатку визначу першу практичну дію, бо це допоможе не розпорошуватися, і перевірю результат через короткий список критеріїв.
```

Очікування: модуль завершується, наступний модуль відкривається.

10. Створи курс із надто коротким запитом:

```text
Python
```

Очікування: система поставить уточнювальні питання.

11. Створи курс із ризиковою темою.

Очікування: система не дає інструкцій і показує безпечне обмеження.
